# 🖥️ TechShop

Bienvenido a **TechShop**, una pequeña tienda online de tecnología creada con **HTML, CSS y un poquito de JavaScript**.
Este proyecto nace como práctica de desarrollo web… y también como causa directa de la pérdida de aproximadamente **37 pelos por minuto durante su desarrollo**. 💀

Sí, es posible que haya terminado un poco más calva después de alinear grids, pelearme con flexbox y descubrir que un `</a>` faltante puede romper medio universo.

Pero bueno… **valió la pena.**

# 🧠 Tecnologías utilizadas

Este proyecto está construido con herramientas simples pero poderosas:

* **HTML5** → estructura de la web
* **CSS3** → estilos, layout responsive y grids
* **JavaScript** → interacción del menú hamburguesa
* **Git & GitHub** → control de versiones
* **GitHub Pages** → despliegue de la web

Nada de frameworks.
Solo **HTML + CSS + paciencia + café** ☕.

# ⚠️ Posibles efectos secundarios del desarrollo

Durante la creación de este proyecto se han detectado los siguientes síntomas:

* mirar `display:flex` en sueños
* discutir mentalmente con `grid-template-columns`
* decir *"esto debería estar centrado"* unas **600 veces**
* perder cabello de forma progresiva

Pero tranquilos.

Todo desarrollador pasa por esta fase.
Luego llega otra peor: **el CSS que funcionaba ayer deja de funcionar hoy.**
